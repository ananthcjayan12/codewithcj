Design Guidelines:
---------------
1. Illustration Style:
   - Flat design vector illustrations
   - Clean, modern tech aesthetic
   - Similar to example image style
   - Relaxed, friendly character illustrations where applicable

2. Color Palette:
   - Primary: Rich Black (#000000)
   - Secondary: Dark Gray (#1A1A1A)
   - Accent: Light Gray (#808080)
   - Details: Clean White (#FFFFFF)
   - No backgrounds (transparent)

3. Visual Elements:
   - Simple geometric shapes
   - Floating UI elements
   - Minimal details
   - Soft rounded corners
   - Light and playful style
   - Monochromatic scheme

4. Technical Specifications:
   - Vector-based illustrations
   - High-resolution export
   - Transparent backgrounds
   - Consistent style across all images
   - Optimized for web

Generation Process:
-----------------
1. Use Midjourney or DALL-E with:
   - Vector style specification
   - Flat design requirement
   - Transparent background
   - Monochromatic black theme
2. Post-process in Illustrator/Figma:
   - Clean up vectors
   - Ensure consistency
   - Add subtle gradients
   - Maintain black and white aesthetic
3. Export formats:
   - PNG with transparency
   - SVG where possible
   - Optimized WebP for web
4. Verify style matches example image with black theme
